# Frontend Mentor - Sunnyside agency landing page

![Design preview for the Sunnyside agency landing page coding challenge](preview.jpg)

Sunnyside Agency Landing Page

Desafio Frontend Mentor

Este projeto é uma solução para o desafio Sunnyside Agency Landing Page da plataforma Frontend Mentor. O objetivo é criar uma página de destino responsiva para uma agência de marketing digital chamada Sunnyside.

Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

Recursos

- Arquivos de design fornecidos pelo Frontend Mentor
- Imagens e ícones utilizados no projeto
- Guia de estilo para ajudar a criar a página

Desafios superados

- Criar uma navegação móvel responsiva
- Implementar efeitos de hover em elementos interativos
- Garantir a compatibilidade com diferentes navegadores e dispositivos
